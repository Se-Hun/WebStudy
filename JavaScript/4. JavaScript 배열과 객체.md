위의 소스코드처럼 선언하면 된다.

{}를 사용한다는 것만 기억하면 된다.

## 배열 안에 객체를 저장하기

    var movies = [
    	{title: "Matrix",
         score: 9.7},
        {title: "Oldboy",
         score: 8.8}
    ]

위의 소스코드처럼 배열 안에 객체를 넣을 수 있다.
